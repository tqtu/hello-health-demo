# API Demo

Simple API testing interface with:
- **Text input** for API endpoint (default: `http://localhost/api/test`)
- **Call API button** to make requests
- **Success/Error alerts** with response data
- **Toast notifications** for feedback

## Quick Start

```bash
npm run dev
```

Open http://localhost:5173

## Usage

1. Enter API endpoint URL in the input field
2. Click "Call API" button
3. View response or error message

## Features

✅ Real-time API testing  
✅ Error handling with messages  
✅ Response display in JSON format  
✅ Loading state with spinner  
✅ Toast notifications  
✅ Responsive design
